Q1. Difference between div and span tag?

The <div> and <span> tags are both HTML tags that are used to group content, but they are used in different ways.
The <div> tag is used to group larger sections of content together, often to apply styles or scripts to them as a group. <div> is a block-level element, which means it takes up the full width of its parent container by default and creates a line break before and after the element.
the <span> tag is used to group smaller sections of content together, often to apply styles or scripts to specific words or phrases. For example, you might use a <span>tag to apply a different color or font size to a specific word within a sentence.<span> is an inline-level element, which means it only takes up as much space as necessary to display its content and does not create a line break before or after the element.

Q2. Difference between class and IDs?

  • The class attribute is used to group elements together that share a common style or function, while the id attribute is used to uniquely identify a specific element on a page.
  • Multiple elements can have the same class, but each element can only have one id and it must be unique on the page.
  • Styles can be applied to all elements with a specific class at once, while styles can be applied to a specific element with an id

Q4. How many types of headings are there ? How to decide which to use where?

In HTML, there are six levels of headings, from <h1> to <h6>. Here's a quick summary of each level:

  • <h1> is the highest level of heading and is typically used for the main title of a page.
  • <h2> is used for major headings within the page, such as section titles.
  • <h3> is used for subheadings within the page, such as subsection titles.
  • <h4> is used for headings within subheadings, and so on.
  • <h5> and <h6> are used for headings at even lower levels of hierarchy, but they are less commonly used.

Q5. Difference between block and inline?

  • Block-level elements create a block of content on the page, while inline-level elements flow within a block-level element. Examples of block-level elements include <p>, <div>, <h1> to <h6>, <ul>, and <li>. Each block-level element starts on a new line.
  • Block-level elements take up the full width available to them, while inline-level elements take up only as much space as necessary to display their content. Examples of inline-level elements include <span>, <a>, <strong>, <em>, and <img>.

Q7. What are attributes?

Attributes in HTML are additional properties that can be added to HTML elements to provide more information about the element. Attributes are used to customize and control the behavior of HTML elements.
Attributes are always included within the opening tag of an element and consist of two parts: the attribute name and the attribute value. The attribute name specifies the type of information being provided, while the attribute value provides the actual information. The attribute value is enclosed within quotation marks.
Some common HTML attributes include src, alt, class, id, style, title, width, height, name, type, and value.

Q8. Difference between HTML tags and elements?

A tag refers to the opening and closing brackets, or angle brackets, used to define an HTML element. For example, <p> and </p> are tags used to define the paragraph element in HTML.
An element, on the other hand, refers to the entire construct between the opening and closing tags, including any content, attributes, or nested elements. For example, <p>This is a paragraph.</p> is a paragraph element that consists of the opening tag <p>, the content "This is a paragraph.", and the closing tag </p>.
